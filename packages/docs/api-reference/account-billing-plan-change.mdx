---
title: "Request account plan change"
description: "Submit a replay-safe Lago-backed Free/PAYG plan change."
openapi: "POST /v1/account/billing/plan-change"
---

Requests a self-service plan change for the active organization. The request is
admin-only and requires an `Idempotency-Key` header.

P1B.18 supports `free` and `payg` as self-service targets. Broader production
enablement is controlled by environment flags until the migration cutover is
complete.
