# Welcome to Prontiq

Prontiq is starting with Australian address validation. The active API surface is the Address API under `/v1/address/*`.

## Quick Start

```bash
curl "https://api.prontiq.dev/v1/address/autocomplete?q=16%20heath%20cres" \
  -H "X-Api-Key: pq_live_..."
```

## Products

| Product                    | Endpoint        | Status      |
| -------------------------- | --------------- | ----------- |
| Address Validation (G-NAF) | `/v1/address/*` | In progress |

ABN, LEI, CVE, and patent APIs are on the roadmap, but they are not part of the active docs until their API routes exist.
