# Welcome to Prontiq

Prontiq is a unified data API platform. One subscription. One API key. One SDK.

## Quick Start

```bash
npm install @prontiq/sdk
```

```typescript
import { Prontiq } from "@prontiq/sdk";

const prontiq = new Prontiq({ apiKey: "pq_live_..." });

const { suggestions } = await prontiq.address.autocomplete({
  q: "16 heath cres",
});
```
