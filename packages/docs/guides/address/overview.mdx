The Address API provides access to the **Geocoded National Address File (G-NAF)** — the authoritative database of 15 million Australian addresses maintained by Geoscape Australia.

## Endpoints

| Endpoint | Purpose | Use case |
|----------|---------|----------|
| [Autocomplete](/guides/address/autocomplete) | Real-time typeahead suggestions | Search bars, forms |
| [Validate](/guides/address/validate) | Verify a full address string | Form submission, data cleaning |
| [Enrich](/guides/address/enrich) | Full record by G-NAF ID | Post-selection enrichment |
| [Reverse](/guides/address/reverse) | Coordinates → nearest addresses | Maps, delivery apps |
| [Postcode Lookup](/guides/address/lookups) | Postcode → localities | Dropdowns, location pickers |
| [Suburb Lookup](/guides/address/lookups) | Suburb → postcodes + bounds | Suburb validation |

## Credits

The Address API uses credits. Endpoint scoring is documented centrally in the [Credits guide](/guides/credits).

## Data coverage

- **15,015,573** addresses across all Australian states and territories
- Every address includes geocode (latitude/longitude), LGA, electorate boundaries, mesh blocks, and ABS statistical areas
- Updated quarterly from the official G-NAF release
- Data source: [data.gov.au](https://data.gov.au/dataset/geocoded-national-address-file-g-naf) (CC BY 4.0)

## Response times

All times are end-to-end (client → API Gateway → Lambda → OpenSearch → response), measured on t3.small OpenSearch (April 2026).

| Endpoint | Warm latency |
|----------|-------------|
| Autocomplete | 150-250ms |
| Validate | 150-220ms |
| Enrich | 150-210ms |
| Reverse | 170-370ms |
| Lookups | 140-320ms |
