---
title: "Reverse Geocode"
description: "Coordinates → nearest Australian addresses with distance."
---

The reverse endpoint finds addresses near a geographic coordinate. Returns results sorted by distance with the distance in meters.

## Basic usage

<CodeGroup>

```bash curl
curl "https://api.prontiq.dev/v1/address/reverse?lat=-33.8568&lon=151.2153&radius=200&limit=3" \
  -H "X-Api-Key: YOUR_API_KEY"
```

```typescript Node.js (server-side)
const res = await fetch(
  "https://api.prontiq.dev/v1/address/reverse?lat=-33.8568&lon=151.2153&radius=200&limit=3",
  { headers: { "X-Api-Key": "YOUR_API_KEY" } }
);
const { results } = await res.json();
```

</CodeGroup>

## Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `lat` | number | Yes | — | Latitude (-90 to 90) |
| `lon` | number | Yes | — | Longitude (-180 to 180) |
| `radius` | number | No | 100 | Search radius in meters (1-50,000) |
| `limit` | integer | No | 5 | Max results (1-20) |

## Response

```json
{
  "results": [
    {
      "id": "GANSW716903927",
      "addressLabel": "SHOP 14, 2 CIRCULAR QUAY EAST",
      "localityName": "SYDNEY",
      "state": "NSW",
      "postcode": "2000",
      "distance_m": 79.2
    }
  ],
  "total": 1
}
```

Each result includes `distance_m` — the distance from the query point in meters. Results are sorted nearest-first.

## Use cases

- **Delivery apps**: Find the nearest valid address to a dropped pin
- **Location verification**: Confirm a GPS coordinate matches an expected address
- **Proximity search**: "What addresses are within 500m of this point?"
