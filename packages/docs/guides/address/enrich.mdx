---
title: "Enrich"
description: "Get boundaries, electorates, geocodes, and mesh blocks for any address."
---

The enrich endpoint returns the complete G-NAF record for an address by its ID. Use the `id` from [Autocomplete](/guides/address/autocomplete) or [Validate](/guides/address/validate) to get the full enrichment data.

## Basic usage

<CodeGroup>

```bash curl
curl "https://api.prontiq.dev/v1/address/enrich?id=GANSW717922337" \
  -H "X-Api-Key: YOUR_API_KEY"
```

```typescript Node.js (server-side)
const res = await fetch(
  "https://api.prontiq.dev/v1/address/enrich?id=GANSW717922337",
  { headers: { "X-Api-Key": "YOUR_API_KEY" } }
);
const address = await res.json();
```

</CodeGroup>

## Enrichment fields

| Category | Fields | Description |
|----------|--------|-------------|
| **Geocode** | `latitude`, `longitude`, `type`, `reliability` | Physical location and geocoding method |
| **Location** | `lat`, `lon` | OpenSearch `geo_point` format |
| **LGA** | `name`, `code` | Local Government Area |
| **Electorates** | `stateElectorate`, `commonwealthElectorate` | State and federal electoral districts |
| **Mesh Block** | `code`, `category` | ABS smallest geographic unit (Residential, Commercial, etc.) |
| **ABS Geography** | `sa2`, `sa3`, `sa4`, `gccsa` | Statistical Areas (name + code) |

## Use case: form enrichment

After the user selects an address from autocomplete:

1. Call **Autocomplete** → user picks a suggestion
2. Call **Enrich** with the `id` → get full record
3. Auto-fill form fields: suburb, state, postcode, coordinates, LGA
