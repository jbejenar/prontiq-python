## Installation

<CodeGroup>

```bash npm
npm add @prontiq/sdk
```

```bash pnpm
pnpm add @prontiq/sdk
```

```bash bun
bun add @prontiq/sdk
```

</CodeGroup>

<Note>
  The SDK is ESM-only. For CommonJS, use `await import("@prontiq/sdk")`.
</Note>

## Setup

```typescript
import { Prontiq } from "@prontiq/sdk";

const prontiq = new Prontiq({
  serverURL: "https://api.prontiq.dev",
  apiKeyAuth: process.env.PRONTIQ_API_KEY!,
});
```

## Autocomplete

```typescript
const { suggestions, total } = await prontiq.getV1AddressAutocomplete(
  "16 heath cres",  // q
  "VIC",            // state (optional)
  5,                // limit (optional)
);

for (const s of suggestions) {
  console.log(`${s.addressLabel} ${s.localityName} ${s.state} ${s.postcode}`);
}
```

## Validate

```typescript
const { match, confidence } = await prontiq.getV1AddressValidate(
  "16 heath crescent hampton east vic 3188"
);

if (confidence === "high") {
  console.log(`Verified: ${match?.addressLabel}`);
}
```

## Enrich

```typescript
const address = await prontiq.getV1AddressEnrich("GAVIC420559144");

console.log(address.geocode);    // { latitude, longitude, type, reliability }
console.log(address.boundaries); // { lga, stateElectorate, meshBlock, sa2, ... }
```

## Reverse geocode

```typescript
const { results } = await prontiq.getV1AddressReverse(
  -33.8568,  // lat
  151.2153,  // lon
  200,       // radius in meters (optional)
  3,         // limit (optional)
);

for (const r of results) {
  console.log(`${r.addressLabel} — ${r.distanceM}m away`);
}
```

## Lookups

```typescript
// Postcode → localities
const { localities } = await prontiq.getV1AddressLookupPostcode("2000");

// Suburb → postcodes
const { postcodes, addressCount } = await prontiq.getV1AddressLookupSuburb(
  "bondi beach",
  "NSW",
);
```

## All methods

| Method | Description |
|--------|-------------|
| `getV1AddressAutocomplete(q, state?, limit?)` | Real-time typeahead suggestions |
| `getV1AddressValidate(q)` | Verify a full address string |
| `getV1AddressEnrich(id)` | Full record by G-NAF ID |
| `getV1AddressReverse(lat, lon, radius?, limit?)` | Coordinates to nearest addresses |
| `getV1AddressLookupPostcode(postcode)` | Postcode to localities |
| `getV1AddressLookupSuburb(suburb, state?)` | Suburb to postcodes |

## Error handling

```typescript
import { SDKValidationError } from "@prontiq/sdk/models/errors";

try {
  const result = await prontiq.getV1AddressAutocomplete("test");
} catch (err) {
  if (err instanceof SDKValidationError) {
    console.error("Validation:", err.message);
  }
  throw err;
}
```
