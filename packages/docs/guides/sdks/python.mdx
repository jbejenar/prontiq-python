## Planned usage

```python
# This is a preview — the package is not yet available
from prontiq import Prontiq

client = Prontiq(api_key="pq_live_...")
suggestions = client.address.autocomplete(q="16 heath cres")
result = client.address.validate(q="16 heath crescent hampton east vic 3188")
```

## Using the REST API now

```python
import requests

response = requests.get(
    "https://59jym47ia1.execute-api.ap-southeast-2.amazonaws.com/v1/address/autocomplete",
    params={"q": "16 heath cres"},
    headers={"X-Api-Key": "YOUR_API_KEY"}
)
data = response.json()
for suggestion in data["suggestions"]:
    print(f"{suggestion['addressLabel']} {suggestion['localityName']} {suggestion['state']}")
```
