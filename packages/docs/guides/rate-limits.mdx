## Monthly quotas

Each tier includes a monthly credit allocation per API family. Usage is tracked per API key, per family, and different endpoints can consume different credit amounts.

| Tier | Credits / Month | Overage | Products |
|------|-----------------|---------|----------|
| **Free** | 10,000 | — | Address |
| **Starter** | 10,000 | $1.50 / 1K credits | All |
| **Growth** | 50,000 | $1.00 / 1K credits | All + SLA |
| **Enterprise** | Custom | Custom | All + support |

## How it works

- Usage is counted per API key, per API family, per calendar month
- Endpoint costs are transparent and expressed in credits, for example Address Enrich costs more credits than Address Autocomplete
- When your quota is exceeded, requests return `429 QUOTA_EXCEEDED`
- The response includes `resets_at` with the next month's start date
- Quotas reset at midnight UTC on the 1st of each month

## API Gateway throttling

In addition to monthly quotas, API Gateway enforces per-route rate limits to protect the infrastructure:

| Limit | Value |
|-------|-------|
| Burst | 100 requests/second |
| Sustained | 50 requests/second |

These limits apply across all API keys. If you need higher throughput, contact us for Enterprise.

<Info>
  Responses already include `X-RateLimit-Limit`, `X-RateLimit-Remaining`, `X-RateLimit-Reset`, and `X-RateLimit-Product` so customers can track remaining monthly credits in real time.
</Info>
