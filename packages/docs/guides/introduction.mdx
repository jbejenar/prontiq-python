## What is Prontiq?

Prontiq is a unified data API for Australian open datasets. One subscription, one API key, one SDK — every product under one roof.

We're starting with **address validation** powered by the G-NAF (Geocoded National Address File) — the authoritative database of every physical address in Australia.

```bash
curl "https://59jym47ia1.execute-api.ap-southeast-2.amazonaws.com/v1/address/autocomplete?q=16+heath+cres" \
  -H "X-Api-Key: your_api_key"
```

```json
{
  "suggestions": [
    {
      "id": "GAVIC420559144",
      "addressLabel": "16 HEATH CRESCENT",
      "localityName": "HAMPTON EAST",
      "state": "VIC",
      "postcode": "3188",
      "confidence": 2,
      "score": 27.72
    }
  ],
  "total": 10000
}
```
