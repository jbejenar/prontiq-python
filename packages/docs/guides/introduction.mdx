## What is Prontiq?

Prontiq is a unified data API for Australian open datasets. One subscription, one API key, one SDK — every product under one roof.

We're starting with **address validation** powered by the G-NAF (Geocoded National Address File) — the authoritative database of every physical address in Australia.

```bash
curl "https://api.prontiq.dev/v1/address/autocomplete?q=9+endeavour+cou" \
  -H "X-Api-Key: your_api_key"
```

```json
{
  "suggestions": [
    {
      "id": "GAQLD156786950",
      "addressLabel": "9 ENDEAVOUR COURT",
      "localityName": "LAMMERMOOR",
      "state": "QLD",
      "postcode": "4703",
      "confidence": 2,
      "score": 25.95
    }
  ],
  "total": 329
}
```

## Credits

Prontiq uses credits as the API usage model.

- Free includes **10,000 credits/month**
- Address API endpoints consume fixed credits per request
- Remaining quota is exposed in response headers

See the [Credits guide](/guides/credits) for the endpoint scoring model.
