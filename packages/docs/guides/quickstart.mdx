## 1. Get your API key

Sign up at the [Prontiq Dashboard](https://d38dvktb4dyib0.cloudfront.net) to get your API key. Keys look like `pq_live_...` for production and `pq_test_...` for sandbox.

## 2. Make your first call

<CodeGroup>

```bash curl
curl "https://59jym47ia1.execute-api.ap-southeast-2.amazonaws.com/v1/address/autocomplete?q=16+heath+cres" \
  -H "X-Api-Key: YOUR_API_KEY"
```

```typescript Node.js (server-side)
const response = await fetch(
  "https://59jym47ia1.execute-api.ap-southeast-2.amazonaws.com/v1/address/autocomplete?q=16+heath+cres",
  { headers: { "X-Api-Key": "YOUR_API_KEY" } }
);
const data = await response.json();
console.log(data.suggestions[0].addressLabel);
// → "16 HEATH CRESCENT"
```

```python Python
import requests

response = requests.get(
    "https://59jym47ia1.execute-api.ap-southeast-2.amazonaws.com/v1/address/autocomplete",
    params={"q": "16 heath cres"},
    headers={"X-Api-Key": "YOUR_API_KEY"}
)
print(response.json()["suggestions"][0]["addressLabel"])
# → "16 HEATH CRESCENT"
```
