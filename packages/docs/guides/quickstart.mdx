## 1. Get your API key

[Contact us](mailto:hello@prontiq.dev) to get your API key. Keys look like
`pq_live_...` for production and `pq_test_...` for sandbox. A self-serve
console-backed account flow is still evolving as the commercial architecture
moves toward Lago.

## 2. Make your first call

<CodeGroup>

```bash curl
curl "https://api.prontiq.dev/v1/address/autocomplete?q=9+endeavour+cou" \
  -H "X-Api-Key: YOUR_API_KEY"
```

```typescript Node.js (server-side)
const response = await fetch("https://api.prontiq.dev/v1/address/autocomplete?q=9+endeavour+cou", {
  headers: { "X-Api-Key": "YOUR_API_KEY" },
});
const data = await response.json();
console.log(data.suggestions[0].addressLabel);
// → "9 ENDEAVOUR COURT"
```

```python Python
import requests

response = requests.get(
    "https://api.prontiq.dev/v1/address/autocomplete",
    params={"q": "9 endeavour cou"},
    headers={"X-Api-Key": "YOUR_API_KEY"}
)
print(response.json()["suggestions"][0]["addressLabel"])
# → "9 ENDEAVOUR COURT"
```
